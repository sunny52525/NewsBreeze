# NewsBreeze
 
NewsBreeze is a News App (made in Jetpack Compose).


## What It has/Tech stack
* UI functionality with Animations
* MVVM Architecture
* Hilt Dependency Injection
* Jetpack compose
* Retrofit
* Coroutines
* Compose Navigation
* Caching using room
* Offline Storage
